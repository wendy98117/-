# -
本小程序以经典的多臂老虎机（Multi-Armed Bandit）问题为例，对强化学习中的 “探索–利用（exploration–exploitation）权衡” 进行可视化演示。假设有多台老虎机，每台机子的中奖概率不同且未知，智能体需要在有限次数内决定每一轮拉哪一台，目标是在长期内获得尽可能高的累计奖励。  在本示例中设置了 4 台老虎机，每台机子的真实中奖概率在 0.2 ~ 0.9 之间随机生成并对玩家保密。每次拉动某台机子会以该概率获得 1 分奖励。 玩家一侧由你手动选择要拉的机子；电脑一侧则采用简单但经典的 ε-greedy 策略： 以 90% 的概率选择当前估计平均收益最高的机子（偏向“利用”），以 10% 的概率随机选择任意机子（保留一定“探索”）。
